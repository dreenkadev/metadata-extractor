# Metadata Extractor

File metadata extraction tool.

Alat ekstraksi metadata file.


## Description

Parses and displays metadata from images, documents, and media files for forensic purposes.

## Deskripsi

Mem-parsing dan menampilkan metadata dari gambar, dokumen, dan file media untuk keperluan forensik.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Extract: python metadata_extract.py image.jpg
- JSON output: python metadata_extract.py document.pdf -o json
- Demo: python metadata_extract.py --demo

## Penggunaan

- Ekstrak: python metadata_extract.py image.jpg
- Output JSON: python metadata_extract.py document.pdf -o json
- Demo: python metadata_extract.py --demo
